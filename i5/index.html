Basic Calculator

A simple calculator for basic mathematical operations.

Instructions:

  • AC - Clear all input
  • DEL - Delete last digit
  • ÷, ×, -, + - Basic operations
  • = - Calculate result

Keyboard Shortcuts:

  • Numbers - Type numbers directly
  • +, -, *, / - Operations
  • Enter - Calculate
  • Backspace - Delete
  • Escape - Clear all
